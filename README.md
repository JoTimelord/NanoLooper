# NanoLooper

## Instructions

Log in to UAF machine and create a work directory

    ssh uaf-10.t2.ucsd.edu
    git clone --recursive https://github.com/sgnoohc/NanoLooper.git
    cd NanoLooper

Set up the environment

    source setup.sh

Compile the code

    cd NanoTools/NanoCORE
    make -j # Compile
